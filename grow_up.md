# 育成ゲーム (grow_up) 機能ドキュメント

`Life-Game-Lab.ino` のメニューから選べる4番目のミニゲーム。育成型のペットゲームで、
実装は [grow_up.h](grow_up.h) / [grow_up.cpp](grow_up.cpp) にある。

## 起動方法

メインメニューで `left`/`right` を押して `Pet` を選び、`select_start` で決定する。
セーブデータが残っている場合はメニュー画面に `CONTINUE` と表示される
([Life-Game-Lab.ino](Life-Game-Lab.ino) の `gameNo == 3` の分岐)。

`grow_up::play()` はブロッキング呼び出しで、内部で無限ループしてメインメニューには
戻らない(Dino・Life Game と同じ設計)。

## 操作

3ボタン共通で以下の役割:

| ボタン | 通常画面 | メニュー画面 |
|---|---|---|
| `left` | アイコンを右へ送る | カーソルを送る |
| `right` | アイコンを左へ戻す | カーソルを戻す |
| `select_start` | アイコンを実行 | 決定 |

ボタンは30msデバウンス付きのエッジ検出で、押した瞬間のみ反応する
(`buttonPressed()`, [grow_up.cpp:793](grow_up.cpp#L793))。

## 成長ステージ

| ステージ | 条件 |
|---|---|
| EGG (たまご) | 開始直後。20秒でふ化 |
| BABY (ベビー) | ふ化後〜3分 |
| CHILD (キッズ) | 3分〜10分 |
| ADULT (アダルト) | 10分〜寿命(60分)まで。ここで姿が4種類に分岐する |
| DEAD (死亡) | お世話ミス10回、または寿命到達で死亡 |

進化のたびに専用メロディが鳴り、進行状況をEEPROMに自動セーブする。

### アダルトの姿(4分岐)

キッズ→アダルトに進化する瞬間、それまでの「お世話ミス回数」「しつけ度」「体重」を
総合判定して姿が決まる([grow_up.cpp:637](grow_up.cpp#L637)):

| 姿 | 条件 |
|---|---|
| ひみつ(星形) | お世話ミス0回 かつ しつけ度50以上 かつ 体重30以下 |
| よい姿(丸型) | お世話ミス3回以下 かつ しつけ度25以上 |
| ふつう(うさぎ耳) | お世話ミス6回以下 |
| わるい姿 | 上記以外 |

## ステータス(内部パラメータ)

すべて `STATUS` アイコンの画面で確認できる。

- **満腹度(FULL)** 0〜100。時間経過で減少。0のまま放置するとお世話ミスが蓄積する
- **ごきげん(HAPPY)** 0〜100。時間経過で減少。20以下でペットが涙を流す
- **体重(WT)** 5〜99。ごはんで+2、おやつで+3、ミニゲームで-2。
  50以上になると「太りすぎ」扱いで病気にかかりやすくなり、画面に `!` が付く
- **しつけ度(DISC)** 0〜100。すねている時に「しかる」で+25。進化の分岐に影響
- **病気(SICK)** かかると満腹度・ごきげんの減少が倍速になる。骸骨アイコンが表示される
- **うんち** 最大4つまで放置される。多いほど病気になりやすい
- **お世話ミス(MISS)** 空腹0/ごきげん0/病気/消灯忘れ(睡眠中)/すね放置が
  一定時間続くたびに1加算。10回で死亡
- **世代(GEN)** / **最長記録(BEST)** 死亡しても引き継がれる歴代記録

満腹度・ごきげんは病気中は減少が倍速、睡眠中は半分の速度になる。

## アイコンメニュー(通常画面)

画面上部の7アイコンを左右で選び、selectで実行する。

| アイコン | 機能 | 拒否される条件 |
|---|---|---|
| ごはん | MEAL(満腹+35)/ SNACK(ごきげん+15・満腹+10)を選ぶサブメニューを開く | 卵/睡眠中/すね中/消灯中。満腹100でMEALを選ぶと首を振って拒否 |
| でんき | 部屋の照明のON/OFFを切り替える | なし |
| あそぶ | ミニゲームを選ぶサブメニューを開く | 卵/病気/睡眠中/すね中/消灯中 |
| くすり | 病気を治療する(下記) | 病気でないとき |
| そうじ | うんちを掃除する | うんちが0個のとき |
| しかる | すねているペットをなだめる/意味なく叱ると悲しむ | 卵/睡眠中 |
| ステータス | ステータス画面を開く | なし |

## 昼夜サイクルと「でんき」

15分周期(`SleepCycleSec` = 900秒)のうち、最初の12分(720秒)が活動時間、
残り3分が睡眠時間になる。睡眠中はペットが `Zz`/`zZ` を表示して動かなくなる。

- 睡眠中に「でんき」で消灯しないと、お世話ミスが加算され続ける
- 消灯すると画面は月と星だけの夜空表示になる
- 朝(活動時間再開)になると自動で点灯し、起床メロディが鳴る
  (`melodyMorning()`, [grow_up.cpp:1484](grow_up.cpp#L1484))

## しつけ(すねる)

キッズ以降、1分ごとに30%(`SulkChancePct`)の確率ですねる。すねると:

- ペットの横に `#` マークが出る
- ごはん・あそぶが拒否される
- 3分(`SulkTimeoutSec`)放置するとお世話ミスが1加算される

「しかる」アイコンで対応すると仲直りしてしつけ度+25。逆に、すねていないのに
「しかる」を使うと無実の子を叱ることになり、ごきげんが-5される。

## ランダムイベント

活動時間中、病気でもすねてもおらず消灯もしていない通常画面で、
1分ごとに12%(`EventChancePct`)の確率で以下のいずれかが発生する
(`doEvent()`, [grow_up.cpp:1158](grow_up.cpp#L1158)):

1. **流れ星** — 画面を星が横切り、ごきげん+10
2. **ともだち来訪** — ベビーの姿の友達が歩いてきてハートを出し、ごきげん+10
3. **プレゼント** — 空からギフトが降ってきて、ごきげん+5・満腹+5

## ミニゲーム

「あそぶ」アイコンから選ぶ。両方とも運動になるため、満腹度-4・体重-2の代償がある。
放置(タイムアウト)すると即座に中断される。

### GUESS(左右当て)

ペットがどちらを向くかを5回勝負で予想する。3勝以上でごきげん+20。

### STOP!(タイミングゲーム)

左右に往復するマーカーを、ランダムに出現するゾーン内で `select` を押して止める。
3回中2回成功でごきげん+20([grow_up.cpp:1037](grow_up.cpp#L1037))。

## 病気とくすり

病気になると満腹度・ごきげんの減少速度が倍になり、太りすぎ・うんち放置・
空腹・不機嫌が原因で発症しやすくなる(判定は1分ごと)。

「くすり」を使うと画面が点滅し、60%の確率で治る。原作再現として、
時々1回では治らず「2回目のくすり」が必要になることがある。

## 死亡と世代交代

以下のいずれかで死亡する:

- お世話ミスが10回(`DeathMistakes`)に達する
- アダルトになってから寿命(60分, `OldAgeSec`)が経過する

死亡すると死亡メロディが鳴り、お墓の画面になる。そのまま `select` を押すと
世代番号(GEN)が+1され、新しいたまごから再スタートする。生存時間が過去の
最長記録を超えていれば `BEST` が更新される。世代・最長記録の記録は
ペットの生死に関わらず永続化される。

## セーブデータ(EEPROM)

Dino のハイスコアが EEPROM の 0〜1番地を使っているため、grow_upは
8番地以降を使用する。

| アドレス | 内容 |
|---|---|
| 8 | マジックバイト (`0xA7`) |
| 9 | セーブバージョン (現在 `2`) |
| 10〜20 | ステージ・姿・満腹度・ごきげん・フラグ(病気/すね/消灯)・うんち数・お世話ミス・年齢(分)・体重・しつけ度 |
| 21 | チェックサム(XOR) |
| 24 | 記録用マジックバイト (`0xB3`) |
| 25 | 世代番号 |
| 26〜27 | 最長生存記録(分) |
| 28 | 記録用チェックサム(XOR) |

- ペットのセーブ(8番地〜)は、進化のたびと5分ごと(`AutoSaveSec`)に自動保存される
- 卵の状態・死亡状態はセーブされない(次回起動時は新しい卵として扱われる、
  または記録のみ引き継がれる)
- 世代・最長記録(24番地〜)はペットが死んでも消えない
- 電源を入れた際 `left` が押されているとgrow_upの
  全セーブ(記録含む)も消去される(`grow_up::clearSave()`)
- セーブバージョンが変わると、旧データはチェック(`SAVE_VERSION`不一致)により
  自動的に無効化され、新しい卵から始まる

## 調整用パラメータ

すべて [grow_up.h](grow_up.h) の定数にまとまっている。

| 定数 | 値 | 意味 |
|---|---|---|
| `HatchSec` | 20秒 | たまごがかえるまで |
| `ChildAgeSec` | 180秒 | ベビー→キッズ |
| `AdultAgeSec` | 600秒 | キッズ→アダルト |
| `OldAgeSec` | 3600秒 | 寿命 |
| `DeathMistakes` | 10回 | 死亡するお世話ミス回数 |
| `MaxPoop` | 4個 | うんちの最大蓄積数 |
| `AutoSaveSec` | 300秒 | 自動セーブ間隔 |
| `SleepCycleSec` / `SleepStartSec` | 900秒 / 720秒 | 睡眠サイクル(12分活動+3分睡眠) |
| `SulkTimeoutSec` | 180秒 | すね放置でミスになるまでの時間 |
| `SulkChancePct` | 30% | 1分ごとにすねる確率 |
| `EventChancePct` | 12% | 1分ごとのイベント発生確率 |
| `FatWeight` | 50 | 太りすぎとみなす体重の閾値 |
